## Возможности

### Генератор паролей

- **Классический пароль** — длина от 4 до 128 символов, выбор наборов (A-Z, a-z, 0-9, спецсимволы)
- **Произносимый пароль** — слоговые паттерны (CV, CVC, VCV), легко запомнить.
- **Исключение похожих символов** — 0, O, o, 1, l, I

### Проверка паролей

- **Анализ стойкости** — визуальная шкала и расчёт энтропии в битах
- **Оценка времени взлома** брутфорсом — от «мгновенно» до «вечность»
- **Проверка утечек** через [Have I Been Pwned](https://haveibeenpwned.com/Passwords) по модели k-anonymity.

## Приватность

**Полная локальность.** Все вычисления происходят в браузере. При проверке утечек через HIBP сервер получает только 5 первых символов SHA-1 хеша, по которым невозможно восстановить оригинальный пароль. Сравнение с утекшими паролями происходит локально.
